## Создано приложение notifications и реализована отправка сообщений

1. Реализовано:

- Создано приложение notifications
- Реализована отправка сообщений в Telegram через Bot API
- Используются переменные окружения (TELEGRAM_BOT_TOKEN)
- Добавлена возможность привязки Telegram-аккаунта пользователя:
    - пользователь сохраняет свой telegram_chat_id
    - данные хранятся в профиле пользователя
- Реализован endpoint для сохранения telegram_chat_id:
    - PATCH /api/users/telegram/

Проведена ручная проверка:

Cообщение успешно отправляется пользователю в Telegram

Логика отправки сообщений вынесена в отдельный сервис (services.py)
