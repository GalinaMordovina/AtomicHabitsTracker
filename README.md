## Реализованы unit- и API-тесты для моделей, валидаторов и эндпоинтов

Запуск тестирования: `pytest --cov`

- Проект покрыт тестами (pytest, pytest-django).
- Общее покрытие кода > 80%.
- Протестированы модели, валидаторы, API и права доступа.

Некоторые служебные методы (str, permission edge cases) намеренно не тестировались, так как не влияют на логику.
