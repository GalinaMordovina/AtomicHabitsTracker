## Настройка CORS и базовой безопасности

В рамках данной ветки добавлены базовые настройки безопасности для API:

- Подключён и настроен пакет django-cors-headers
- Добавлен CorsMiddleware в корректном порядке middleware
- Настроена обработка CORS-запросов для клиентских приложений
- Проведена финальная проверка settings.py на корректность конфигурации

Использован безопасный и распространённый подход `CORS_ALLOW_ALL_ORIGINS = True`. Конфигурация легко ужесточается при необходимости (через `CORS_ALLOWED_ORIGINS`)

Результат:

- API корректно принимает запросы из браузера
- Swagger / Redoc работают без CORS-ошибок
- Конфигурация безопасности прозрачна и понятна
